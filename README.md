# Simple Interest Calculator

## Project Overview
This project provides a Bash script for calculating simple interest based on the standard financial formula:

**Simple Interest Formula:**  
`I = P × R × T / 100`

Where:
- **I** = Interest amount
- **P** = Principal amount (initial investment)
- **R** = Annual interest rate (percentage)
- **T** = Time period (in years)

## Features
- Interactive command-line interface
- Input validation for numeric values
- Floating-point precision calculations
- Clear output formatting

## Script Usage

### Prerequisites
- Bash shell (version 4.0+)
- `bc` calculator utility (for precise decimal calculations)

### Running the Calculator
```bash
# Make the script executable
chmod +x simple-interest.sh

# Run the calculator
./simple-interest.sh
